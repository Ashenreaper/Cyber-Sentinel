# Cyber-Sentinel
Artificial Intelligence Powered Cyber Defense System for Detecting Web Application Attacks
This project implements an Artificial Intelligence based cybersecurity system that detects and mitigates web application attacks.

Technologies used:

- Kali Linux
- Python
- PostgreSQL
- Docker
- Scikit-learn

Main components:

- Attack Simulation
- Log Collection
- Database Storage
- Machine Learning Detection
- Alert System
